# Backend
* Dev env:
  * mvn clean install [-DSkipTests]
  * java -jar <generated jar>
# Frontend
* Dev env: 
  * npm i; 
  * npm run build
  * npm run start
* Staging env:
  * npm i;
  * npm run build-stag
  * npm run start-stag
* Production env:
  * npm i;
  * npm run build-prod
  * npm run start-prod
